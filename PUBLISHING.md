# PUBLISHING

## Pre-publish flow
1. `./.github/skills-config/ebook-build/invoke-build.ps1` を実行する
2. 以下の成果物を確認する
   - `ebook-output/clean-architecture.epub`
   - `ebook-output/clean-architecture.pdf`
   - `ebook-output/clean-architecture-kdp-registration.md`
3. `README.md` / `docs/index.md` / `MASTER-INDEX.md` の導線を確認する
4. 章構成・サンプルコード・KDP 登録情報の整合性を確認する
5. GitHub Pages と配布先を更新する
